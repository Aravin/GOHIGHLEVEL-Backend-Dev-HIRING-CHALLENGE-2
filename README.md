## Analysis & Design

### Tech Stacks Choosed

#### Frontend

* Next.js (React.js based framwork)
* Tailwindcss with Daisy CSS (one of the best combination)
* LocalStorage for Session
* React Data Table for Hisotory Page

#### Backend

* Node.js (Runtime)
* TypeScript
* MongoDB

#### Hosting

* Vercel for Frontend
* Heroku for Backend
* Atlas for MongoDB
